# SHADOW QUIZ

# ADS
- 페르소나 : 20대 대학생, 지루한 지하철/통학버스 시간을 즐겁게 보내고 싶어하며, 경쟁적인 요소가 있는 게임을 통해 성취감을 느끼는 것을 선호한다.
- 문제점 : 킬링 타임용 무료게임은 과금 유도가 많아 만족스러운 경험을 얻기 어렵다.
- 해결방안 : 경쟁심을 자극하고 성취감을 제공하며, 과금 없이 쾌적하게 킬링 타임을 즐길 수 있는 무료 게임 앱을 개발한다.

# 주요기능
- 간단하고 직관적인 UI/UX
- 경쟁적인 랭킹 시스템
- 짧은 플레이 시간을 위한 빠른 진행
- 다양한 게임 모드 (난이도, 주제 등)

# 작동환경
- XCODE 16.4
- iOS 18.5
